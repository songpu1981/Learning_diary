
#Development Environment

##Use “eclipse" and “ADT" 
  1. Install the latest Java version(e.g. jdk8u25)
  2. Download the latest stand-alone Android SDK tool
  3. Update the Android SDK
  4. Install the [“Intel Hardware Accelerated Execution Manager"](https://software.intel.com/en-us/android/articles/intel-hardware-accelerated-execution-manager/)
  5. Create a virtual device
  6. Download the latest eclipse 
  7. In eclipse, install the “Android Development Tool”
  8. In eclipse, set up the configuration
